1.1 Starting Logging

StartLogging( fileName, level, [maxFiles, maxFileSize, prefix])
Start logging to the specified file at the given level.

StartLoggingForThread( fileName, level, [maxFiles, maxFileSize, prefix])
Start logging to the specified file at the given level but only for the current Python thread.

StartLoggingStderr( level, [prefix])
Start logging to stderr at the specified level.

StartLoggingStdout( level, [prefix])
Start logging to stdout at the specified level.
